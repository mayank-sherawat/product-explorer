# Product Explorer

Full-stack product exploration platform powered by live scraping of World of Books.

## Tech Stack

### Backend
- NestJS
- Prisma
- PostgreSQL
- Crawlee + Playwright

### Frontend
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Features
- Navigation → Collections → Products → Product Detail
- Live scraping with caching
- Product detail enrichment
- PostgreSQL persistence
